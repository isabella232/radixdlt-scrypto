pub struct Track<'s, S: SubstateStore> { /* fields omitted */ }
Expand description

An abstraction of transaction execution state.

It acts as the facade of ledger state and keeps track of all temporary state updates, until the commit() method is called.

Typically, a track is shared by all the processes created within a transaction.
